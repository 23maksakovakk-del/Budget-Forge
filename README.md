Budget Forge — Бюджетный менеджер на 7 языках
Budget Forge — это коллекция из семи независимых реализаций интеллектуального бюджетного менеджера, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для управления личными финансами с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
📊 Управление доходами и расходами: добавление транзакций с категорией, суммой, датой и описанием.

🎯 Бюджетирование по категориям: установка лимитов на месяц (или произвольный период) и отслеживание остатка.

📈 Статистика и отчёты:

Общая сумма доходов/расходов за период.

Траты по категориям с процентами.

Прогресс выполнения бюджета (остаток/перерасход).

💾 Сохранение и загрузка данных в JSON.

📤 Экспорт/импорт в CSV и TXT.

🔔 Предупреждения при превышении бюджета (в CLI/GUI/веб).

🌍 Поддержка валют (настраиваемый символ).

🖥️ Интерфейсы:

Командная строка (CLI) с интерактивным меню.

Графический интерфейс (GUI) в Python, Java и C#.

Веб-интерфейс в PHP.

Использование (общий синтаксис)
CLI (все версии):

bash
budget_manager --add --type income --category Salary --amount 1500 --date 2024-01-15
budget_manager --set-budget --category Food --amount 500 --month 2024-01
budget_manager --list --type expense --from 2024-01-01 --to 2024-01-31
budget_manager --stats
budget_manager --export --output report.csv
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
